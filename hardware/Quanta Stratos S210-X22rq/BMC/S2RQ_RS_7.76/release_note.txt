On-Going item:

Known Issues:
	
Notice:
	0. This version is for ME which get power consumption from PSU directly.
	a) BMC will not issue PSU predictive fail SEL by itself.
	b) ME will issue PSU related log if PSU support PMBus V1.2

	1. Update BMC FW via command line: 
	a) put dos4gw.exe, socflash.exe, and rom.ima in same directory.
	b) run "dos" or "update_all.bat" in DOS

	3. login account: 
		1) web and ipmitool 
			username: admin 
			password: admin
		2) SMASH
			username: sysadmin
			password: superuser
	4. Single NIC enable in this version. Default NIC is dedicated NIC.
	
	5. Don't use WebUI update with reserving configuration on original BMC Version
	   before V3.22. 
	a) We change the ROM image partition after V3.22.
	b) If some feature abnormal after Web UI update. Please update again without 
	   reserving configuration.
	c) If BMC cannot work, please refer Notice 1.e to update again with BMC disable.
